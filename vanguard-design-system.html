Key Typography Corrections from Previous Version
Font family: Confirmed as "FF Mark" (not Helvetica Neue). This is a licensed commercial font ‚Äî use Inter or DM Sans as a prototype substitute.
Primary text color: #040505 (not #1a1a1a) ‚Äî near-black with a very slight warm tint.
Table column headers: Sentence case at 12px/600 weight (not uppercase). The uppercase treatment is used only for section eyebrows and some labels, not table headers.
Primary CTA button: Near-black #040505 with pill radius (not red). Red (#C8102E) is reserved for brand/marketing and active state indicators ‚Äî not transaction CTAs.
Link color: #1255CC (not #1a6aaa) ‚Äî confirmed from input focus border and "Transfer cash" link.
